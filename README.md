# Посмотри в окно

Проектная работа по курсу веб-разработки Яндекс Практикума.
https://github.com/Nikolai-Zhuravlev/posmotri-v-okno-fd

## О проекте

«Посмотри в окно» — одностраничное приложение, которое показывает прямые трансляции и записи из окон разных городов мира. Можно выбрать город и время суток, чтобы найти подходящее видео.

## Функциональность

- Загрузка видеокарточек из внешнего API при открытии страницы
- Поиск по городу и времени суток
- Переключение видео кликом по карточке
- Подгрузка дополнительных карточек кнопкой «Показать ещё»
- Прелоадеры на время загрузки данных
- Обработка ошибок

## Технологии

- HTML5
- CSS3 (Flexbox, псевдоэлементы, кастомные чекбоксы)
- JavaScript (Fetch API, работа с шаблонами `<template>`)

## Запуск

Клонировать репозиторий и открыть `index.html` в браузере.
